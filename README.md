# Enterprise Content Intelligence Agent

**Track**: Enterprise Agents - AI Agents Capstone Project

## Overview

Multi-agent system that reduces enterprise research time by 70% through intelligent orchestration of specialized AI agents.

## Features

- ✅ Multi-agent orchestration (5 specialized agents)
- ✅ Parallel & sequential execution patterns
- ✅ Session management & long-term memory
- ✅ Context engineering
- ✅ Full observability
- ✅ Interactive demo

## Quick Start
```bash
# Install dependencies
pip install -r requirements.txt

# Add your API key to .env
cp .env.example .env
# Edit .env with your GOOGLE_API_KEY

# Run demo
open demo.html
```

## Repository Contents

- `demo.html` - Interactive React demo
- `requirements.txt` - Python dependencies
- `simple_agent.py` - Simplified agent implementation
- `test_agent.py` - Basic functionality tests

## Demo

Open `demo.html` in your browser to see the interactive multi-agent system in action.
